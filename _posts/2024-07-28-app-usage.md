## 镜像

给各个 linux 发行版配置官方镜像

### ubuntu

```bash
sudo sed -i "s@//.*archive.ubuntu.com@//mirrors.huaweicloud.com@g" /etc/apt/sources.list.d/ubuntu.sources


#ubuntu ports 适用于arm版本
sudo sed -i "s@//.*ports.ubuntu.com@//mirrors.huaweicloud.com@g" /etc/apt/sources.list

```
